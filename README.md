# 🔒 AI-Powered Email Security Analyzer

A **cybersecurity tool** that analyzes email headers and content to detect **phishing, spoofing, malicious links, suspicious domains, and dangerous attachments** using automated threat analysis.

---

## 🚀 Features

* 📧 Email Header Analysis (SPF, DKIM, DMARC validation)
* 🌐 Domain & Sender Security Checks
* 💬 AI-based Social Engineering Detection
* 🔗 Malicious Link & Redirect Detection
* 📎 Dangerous Attachment Identification
* 📊 Automated Risk Score & Threat Classification
* 🖥️ Simple Web Interface for Email Analysis

---

## 🛠️ Technologies Used

* **Python**
* **Flask**
* **Transformers (AI / NLP)**
* **BeautifulSoup**
* **dnspython**
* **python-whois**
* **HTML, CSS, JavaScript**

---

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/AI-Email-Security-Analyzer.git
cd AI-Email-Security-Analyzer
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python email_security_analyzer.py
```

Open in browser:

```
http://localhost:5000
```

---

## 📊 Output

The tool generates:

* Email **Risk Score (0–100)**
* Threat **Classification**
* Detected **Security Indicators**
* Detailed **Threat Analysis**
* **Security Recommendations**
